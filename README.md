# search4letters.web
Searches for the specified characters in the entered word.
===========
- Test web app. 
- Created using **Flask**.
- Query logging is supported (file vsearch.log + MySQL).
- *vsearch4web_v6.py* is the latest version of the application.
# Templates
- static - css template.
- template - html template.
